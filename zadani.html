Základy JS - opakování

1: Co je JavaScript a k jakým účelům se používá?

Poskytněte vhodné odpovědi a použijte nečíslovaný seznam.

V textu použijte alespoň 3 internetové odkazy na zajímavé příklady použití JS!

  • JavaScript je vysokoúrovňový, interpretovaný skriptovací jazyk.
  • Používá se k přidání interaktivity a dynamických prvků na webových stránkách.
  • Syntax JavaScriptu je podobná jiným programovacím jazykům, jako jsou C, C++ a Java, což usnadňuje učení a používání.
  • JavaScript umožňuje manipulaci s webovými stránkami a dokumenty, včetně úpravy obsahu, přidávání událostí a reakci na uživatelské interakce.
  • Kromě webových prohlížečů se JavaScript používá také na straně serveru prostřednictvím platformy Node.js, což umožňuje psát serverový kód v JavaScriptu.
  • Existuje mnoho knihoven a frameworků, které usnadňují vývoj v JavaScriptu, jako například React, Angular, Vue.js pro tvorbu uživatelských rozhraní a Express.js pro tvorbu serverového kódu.
  • JavaScript je podporován většinou moderních prohlížečů a není omezen na konkrétní platformu, což umožňuje psát kód, který běží na různých zařízeních a prohlížečích.

2: Jaká pravidla platí pro psaní kódu v JavaScriptu?

Napiš aspoň pět zásad ve podobě číslovaného seznamu.

Použij angličtinu!

  1. Ensure consistent and clear indentation in your code to enhance readability. A commonly accepted practice is to use 2 or 4 spaces for each level of indentation.
  2. Employ meaningful names for your variables and functions, making them self-explanatory and easier to comprehend.
  3. Incorporate comments to elucidate intricate or less obvious parts of your code. Well-placed comments aid other developers (including your future self) in understanding your code.
  4. Reduce the usage of global variables to mitigate naming conflicts and maintain data encapsulation. Whenever possible, utilize local or function-scoped variables.
  5. Adhere to a consistent coding style, whether it's your personal style, your team's style, or a recognized style guide such as the "JavaScript Standard Style" or the "Airbnb JavaScript Style Guide." This practice ensures consistency and readability throughout your codebase.

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // s přiřazením prázdné (žádné) hodnoty
let cele = 5; // celé číslo (typ number)
let realne = 7.5; // reálné číslo (typ number)
let desetinne = 5e2; // zápis desetinného čísla pomocí exponentu
let text = "Ahoj"; // typ string
let prestupny = false; // proměnná typu boolean, nabývá pouze hodnot true nebo false

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

=, +=, -=, /=, *=, %=, ++, --, += (pro spojování řetězců)

Logické

&& (a zároveň), || (nebo), ! (negace), != (nerovná se), ? : (ternární operátor)

Porovnávací

== (rovná se), === (strikně se rovná, posuzuje i datový typ)
Pro porovnávání čísel:
< (menší), <= (menší nebo rovno), > (větší), >= (větší nebo rovno)
Větší a menší se také používá k porovnávání lexikografického pořadí

Bitové

&, |, ~, ^, <<, >>, >>>

Typové

typeof (typ proměnné), instanceof (), cosntructor

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

// Zobrazí se vyskakovací okno se zprávou Nezadal jsi správné heslo!
alert('Nezadal jsi správné heslo!');

prompt()

// Do proměnné password vloží heslo zadané uživatelem
let password = prompt('Zadej heslo:');

confirm()

// Zobrazí se vyskakovací okno, které se uživatele ptá, jestli chce přesměrovat na stránku seznam.cz
if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';

Varování:

  1. Prohlížeč může mít blokovací systém který tyhle okna zablokuje
  2. Stránky s vyskakovacími okny nebudou vypadat důvěryhodně.
